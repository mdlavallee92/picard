# Build Study Hub

Build Study Hub

## Usage

``` r
buildStudyHub(projectPath = here::here(), previewHub = TRUE)
```

## Arguments

- projectPath:

  the path to the Ulysses repo, by default takes the path of the active
  R project

- previewHub:

  toggle to preview the hub after it builds. Default is TRUE

## Value

invisible return. Creates \_site folder with html files to preview site
